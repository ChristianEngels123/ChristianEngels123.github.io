Asteroid Defence Game: CMPG 323 Project

This was a group assignment in my 3rd year of B.Sc Information Technology. The goal of this assignment was to work under an Honours student who served as our project manager, and to create whatever program the Honours student identified. In this case it was a tower defence game revolving around Asteroids. We had to go through the entire development process as part of the project and deliver the final game in one semester.

Technologies Used:
• Godot Game Engine
• Sound Editing Software
• Image Editing Software
• Full Development Lifecycle (Design, Development, Testing, Deployment)
